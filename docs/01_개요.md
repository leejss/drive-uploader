# Drive Uploader 개요

## 프로젝트 소개

Drive Uploader는 터미널에서 구글 드라이브에 파일과 폴더를 쉽게 업로드할 수 있는 커맨드라인 도구입니다. Go 언어로 개발되었으며, 간단한 명령어로 구글 드라이브와의 상호작용을 가능하게 합니다.

## 주요 기능

### 파일 업로드
- 단일 파일 업로드 지원
- 폴더 단위 업로드 지원
- 재귀적 폴더 구조 유지

### 인증 관리
- OAuth2 기반 구글 드라이브 인증
- 다중 프로필 지원 (계획 중)
- 간편한 인증 상태 관리
  - `auth list` - 인증된 계정 목록 조회
  - `auth login` - 새 계정 로그인
  - `auth logout` - 계정 로그아웃

### 사용자 경험
- 직관적인 커맨드라인 인터페이스
- 상세한 진행 상황 표시
- 에러 발생 시 자동 복구 절차

## 개발 목적

이 프로젝트는 다음과 같은 목적을 가지고 있습니다:

1. **실용성**: 개발자와 일반 사용자 모두를 위한 간편한 구글 드라이브 업로드 도구 제공
2. **학습**: Go 언어의 실제 프로젝트 적용 및 커맨드라인 프로그램 개발 경험
3. **아키텍처**: 확장 가능하고 유지보수가 용이한 코드 구조 설계
4. **오픈소스**: 커뮤니티 기반의 지속적인 개선과 기여

## 기술 스택

- **언어**: Go 1.24.0
- **주요 라이브러리**:
  - `google.golang.org/api/drive/v3` - 구글 드라이브 API
  - `golang.org/x/oauth2` - OAuth2 인증
  - `github.com/joho/godotenv` - 환경 변수 관리
- **아키텍처**: Clean Architecture 패턴 기반
